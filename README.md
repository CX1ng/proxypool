### Golang IP代理池

+ 安装  
基于Golang 1.9开发
```
go get github.com/CX1ng/proxypool
```
+ 安装依赖  
使用Golang依赖管理工具[glide](https://github.com/Masterminds/glide)进行项目依赖管理
```
glide install
```
